# dope-monorepo

Nouns DAO is a generative avatar art collective run by a group of crypto misfits.

## Contributing

If you're interested in contributing to Nouns DAO repos we're excited to have you. Please discuss any changes in `#developers` in [discord.gg/nouns](https://discord.gg/nouns) prior to contributing to reduce duplication of effort and in case there is any prior art that may be useful to you.

## Packages

### contracts

The [contracts](packages/contracts) is the suite of Solidity contracts.

### web

The [web](packages/webapp) is the frontend for interacting with the web app.

## Quickstart

### Install dependencies

```sh
yarn
```

### Run webserver for development

```sh
yarn web:dev
```

### Build all packages

```sh
yarn build
```

### Run Linter

```sh
yarn lint
```

### Run Prettier

```sh
yarn format
```
